
```
; ;.*\n          ->  空   删除匹配行
 
\s*\n[\s]*\n     -> \n    删除空行

\[              ->  \n[   指定符号前插入空行
```




mac 快捷键

移动到单词的最前面：option + ←
移动到单词最末尾：option + →
将当前行代码移动到上一行：option + ↑
将当前行代码移动到下一行：option + ↓
移动到当前行最前面：cmd + ←
移动到当前行最末尾：cmd + →
花括号之间跳转：cmd + shift +
移动到文档第一行或最后一行：cmd + ↑ / cmd + ↓



代码回去
option + command + <
option + command + >
