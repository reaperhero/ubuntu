# 语法

```
-> 操作符用于类、对象的操作
$user->friends_count  取对象的friends_count属性。
$user->homeTimeline($p)  调用对象的homeTimeline方法，方法中传入一个参数$p

=>操作符通常用于数组
$css=array('style'=>'0'，‘color’=>‘green‘) 定义数组键名让它指向数组末一个元素

:: 静态方法和静态属性的引用方法
Test::$test 获取Test类$test属性的值
Test::test(） 直接调用Test类静态方法test 
```
